# Syntecxhub_Project_LibraryBookInventoryManager_Sakshi

## Description
A simple Library Book Inventory Manager developed using Python.

## Features
- Add Book
- Search by Title
- Search by Author
- Issue Book
- Return Book
- Library Report
- JSON File Storage

## Concepts Used
- Python OOP
- Classes and Objects
- Dictionary (HashMap)
- JSON File Handling
- Loops and Conditional Statements

## How to Run

```bash
python main.py
```
